Проект Zoo Manager - форма лля отслеживания животных в каких городах. Свободное использования и очень удобное. 

Что может?

Показывает всех животных ы зоопарке
Показывает всех животных в выделенном зоопарке
Отслеживает какие в наличии есть животные 
Удалять зоопарк / добовлять зоопарк 
Удалять животного из базы данныз / Добавлять
Обновляет название зоопарка/ Животных

Запуск формы.

Скачивания -> Заапуск через IDL -> Загрузку ждем  -> Ctrl+F5 -> И работаем 


Функции формы.

private void ShowZoos() - показ всех животных в зоопарке 
private void ShowAssociatedAnimals() - показ всех живоных в выделенном зоопарке
private void listZoos_SelectionChanged() - отслеживания какте животные в зоопарке
private void ShowAllAnimals() - показ всех животных которые есть в базе данных
private void DeleteZoo_Click() - удаляет зоопарк 
private void AddZoo_Click() - добавляет зоопарк
private void AddAnimalToZoo_Click() - добавляет животного в зоопарк 
private void AddAnimal_Click() - добавляет животного в список
private void DeleteAnimal_Click() - удаляет животного из списка всех животных
private void UpdateZoo_Click() - обновляет название зоопарка при нажатии на зоопарк
private void UpdateAnimal_Click() - обновляет название животного при нажатии на животного находящегося в списке всех животных
